## Basic Operations
- addition (`+`) 
- substraction (`-`)
- multiplication (`*`) 
- division (`/` ---> returns value with floating point 1) 
- interger division (`//` ---> returns interger value)
- exponentiation (`**`), modulo (`%`)

# String
- `"value"`
- `'value'`

## Print Function
- `print(value)` ---> for numberic values
- `print('value')` or `print("value")` ---> for strings
- `print(r'value')` ---> raw value: prevents special cases like `\t`, `\n`, etc.
- `print(f'value')` ---> formatted string literal: provides concise & readable way

## Printing Quote Marks
- `print("'value'")` ---> print ''
- `print('"value"')` ---> print ""
- `print("value\'s and "value"")` ---> print both "" and '

## String Concatenation & Operations
- `'value'` + `'value'` ---> returns valuevalue
- `3 * 'value'` ---> returns valuevaluevalue

# Shell
- use `python3` to enter shell scripting
- use `help` for help
- in help> 
1. type topics to explore modules included in python
2. type `MODULE_NAME` to get information e.g.: LISTS
3. type `quit` to get back to python shell
- directy use `help('MODULE_NAME')` to get information of a module
